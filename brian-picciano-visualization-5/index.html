Visualization 5

2021-05-28 by Brian Picciano - (4 min read)

This visualization combines two distinct layers, each of them borrowing their behavior from Visualization 4. Neither layer has any effect on the other, one is merely super-imposed on top of the other in the top canvas. You can see each layer individually in the two lower canvases.

Despite their not affecting each other, the code is set up so that each layer could be affected by the other. This will likely be explored more in a future post.
